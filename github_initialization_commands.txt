First Time github setup
==============================
1. git init
2. git add .
3. git commit -m "any message"
4. git remote add origin https://github.com/jasraj9302/property-management.git
5. git remote set-url origin https://jasraj9302:{MY_TOCKEN}/jasraj9302/property-management.git
6. git push --set-upstream origin master
